# ny_taxi_rides dbt project

This is a clean starter dbt project for the NYC taxi rides pipeline.

## Structure

- `models/staging/` — staging models and source definitions
- `models/marts/` — final analytics tables
- `seeds/` — small lookup or reference data
- `macros/` — reusable SQL macros
- `tests/` — schema or data tests
- `snapshots/` — snapshot definitions
- `analyses/` — exploratory analyses
